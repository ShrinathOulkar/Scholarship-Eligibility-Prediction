# Scholarship-Eligibility-Prediction
🎓 Scholarship Eligibility Prediction
📌 Project Overview

The Scholarship Eligibility Prediction System is a Machine Learning-based application developed to predict whether a student is eligible for a scholarship based on various academic, financial, and demographic factors.

The system analyzes student data and uses classification algorithms to determine eligibility status (Eligible / Not Eligible). This project helps automate the scholarship screening process and supports fair and data-driven decision-making.

🚀 Key Features

Student data preprocessing and cleaning

Handling missing values and outliers

Feature selection and encoding

Model training using classification algorithms

Performance evaluation (Accuracy, Precision, Recall, F1-Score)

Bias and overfitting analysis

Real-time prediction using Streamlit interface

🧠 Machine Learning Techniques Used

Supervised Learning

Classification Algorithms (e.g., Random Forest, Logistic Regression, Decision Tree)

SMOTE for handling class imbalance

Train-Test Split

Hyperparameter Tuning
